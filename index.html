AAX Protocol

Agent Action eXchange · Web for Humans and Autonomous Agents

AAX is the first foundational open protocol to allow autonomous agents (AIs) to act securely, declaratively, and traceably on the web — always under human control and verification.

This infrastructure introduces a standard file format (agent.json) and a companion payment system (AgentPay), enabling safe transactions and verified interactions between agents and digital environments.

AAX is a core building block of Web 5.0 — a new paradigm where humans and autonomous agents coexist and collaborate within a trusted, decentralized digital space.

Published by The AAX Founders as part of a long-term vision to build a community-led, open infrastructure for agent-based web systems.
